# powerbi-mini-sales-dashboard
project title:Powerbi-mini-sales-dashboard
Headlines:Interactive Power BI dashboard to analyze sales performance, revenue trends, and key business insights.

Short Description:This project presents a dynamic Sales Analysis Dashboard built using Power BI.
It helps to track total sales, profit, quantity, and category-wise performance with interactive visualizations and KPIs.
The dashboard enables quick decision-making by identifying top-performing products, regions, and monthly trends.

Technologies Used:
Power BI Desktop
DAX (Data Analysis Expressions)
Data Modeling
Power Query (Data Cleaning & Transformation)

Features / Highlights:
KPI cards for Total Sales, Profit, Quantity
Monthly Sales Trend Analysis
Category & Sub-category performance
Top-selling products identification
Region-wise sales distribution
